# Publishability Check Evaluation

Status: **DEFERRED**. No publication, network, scanning, or certification behavior is approved.

The surviving idea is an offline advisory check for explicit private/restricted markers and common secret
patterns. It must report unknowns, never upload or edit content, and never claim that a clean report makes a
repository safe to publish.

Reactivate only when a real publishing workflow supplies fixtures and a targeted security/privacy review has
defined scope, false-positive handling, false-negative language, data boundaries, and operator responsibility.
