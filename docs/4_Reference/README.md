# Reference Documents

This folder holds source reports, market research, design notes, and competitor evidence that inform
Memory Seed planning but are not active implementation proposals.

Lifecycle:

- `docs/1_Inbox/` is for unassessed incoming material.
- `docs/2_Todo/` is for refined active plans.
- `docs/2_Todo/completed/` is for shipped, rejected, superseded, or otherwise resolved plans.
- `docs/3_Spec/` is for live normative contracts and audits that current code/plans rely on.
- `docs/4_Reference/` is for source-only research that should remain available as provenance.

When a reference document produces actionable work, promote the actionable scope into a focused
`docs/2_Todo/*.md` plan and keep the source document here.
