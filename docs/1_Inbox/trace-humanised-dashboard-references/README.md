# Humanised Memory Trace visual references

Status: **source capture and mockup exploration; not an approved design spec**.

## Reference themes

The captured images consistently suggest:

- warm cream and parchment surfaces instead of neutral grey application chrome;
- editorial serif headlines paired with quiet, legible sans-serif utility text;
- olive, ochre, terracotta, dusty blue, and faded green accents;
- large margins and deliberate asymmetry;
- brand-system restraint: a small palette, strong typography, and few decorative effects;
- a human, cultural feel without turning the product into lifestyle imagery.

## Translation into Trace

Use these references for tone, hierarchy, palette, and spacing—not for direct copying. Trace must preserve
accessible contrast, exact relationship semantics, dense evidence inspection, keyboard navigation, and the
three-region workspace. Avoid Instagram chrome, photographic backgrounds, ornamental illegibility, or
hand-drawn paths whose geometry obscures provenance.

## Mockup directions

1. **Field Notes** — calm research notebook; timeline-led; evidence in a readable margin.
2. **Living Archive** — more spatial and narrative; day groups become chapters and the inspector becomes a
   document reading surface.
3. **Editorial Focus** — reduced chrome and progressive disclosure for users who primarily read and review
   decisions rather than administer the corpus.

Final generated mockups should be stored beside this file with descriptive names once selected.
