### Summary

- Implemented Phase 0 of the approved Memory Trace roadmap: the optional `branch:` entry-YAML scalar,
  captured at record time, so branch history starts accruing before the Trail view (Arc 2) exists.
  This entry dogfoods it: `branch: main`.

### Decision

- D: Add `branch: str | None` to `MemoryChunk` (parsed via `_metadata_value` at all three construction
  sites), surface it read-only in the shared retrieval service (`ranked_to_dict`/`chunk_to_dict`) and
  the Lense API mapper (`_chunk_to_api`), document it in `session_logging.md` (solo capture via
  `git rev-parse --abbrev-ref HEAD`) and `agent_collaboration.md` Final Handoff Gate (orchestrated
  backfill from Task Packet `working_branch`), and in `docs/graph-edge-contract.md` as a stored label.
- R: A branch is a durable record-time fact like a commit SHA; deriving it later from git is unstable
  (branches move/delete/rebase). Shipping the field first lets data accumulate so the eventual Trail
  view has real history to render.
- A: Considered adding a `branch` plausibility check to `check_session_links()` (the plan's wording);
  rejected - `branch` has no referential integrity, links check ignores unknown YAML keys, and an
  empty `branch:` already reads as absent (`_metadata_value` returns `value or None`). Leaving the
  validator untouched keeps the "never queries git" guarantee trivially and minimizes blast radius.
  Also rejected a `worktree:` field (ephemeral local path, no evolution semantics) - documented as a
  standing non-goal in the schema doc and graph-edge contract.
- F: `memory_seed/semantic_cache.py` (MemoryChunk + 3 sites), `memory_seed/retrieval.py`,
  `memory_seed/lense.py` (_chunk_to_api), `.memory-seed/skills/session_logging.md` (+ seed twin),
  `.memory-seed/skills/agent_collaboration.md` (+ seed twin), `docs/graph-edge-contract.md`,
  `tests/test_retrieval.py`.
- T: Full suite green (258 passed, 7 subtests); new `test_branch_field_parses_and_surfaces_read_only`
  proves parse + read-only surfacing (service == MCP `memory_get_chunk`) + links-check stays clean and
  git-free. `links check` OK (27 files); `doctor` healthy. Seed twins byte-identical. Version bump held
  pending an explicit release decision (branch data has no consumer until Arc 2).
