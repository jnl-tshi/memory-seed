### Decision

- D: First slice of JNL's design-feedback batch (stability, edge noise, resizable panes, humanist
  light/dark theme, hand-drawn Trail - direction saved to agent memory as `trace-design-language`).
  Rebuilt `GraphWorkspace`'s lifecycle: the Cytoscape instance now mounts **once per graph payload**;
  selection ring, label policy, and evolves-edge reveal apply **in place** via a shared
  `applyPresentation` (cy.batch data/class updates, called by both the async mount and the update
  effect, reading current state from refs). The rendered element set is **selection-independent**
  (every node with any edge always renders; evolves edges hide/reveal via a style class, never element
  churn). Layout is deterministic: nodes get hash-ordered initial positions on a circle
  (community+id sort), then `cose` with `randomize: false` - same data always settles to the same map.
  Edges are now `curve-style: straight` (was unbundled-bezier with a 38px bow on every edge). Instance
  exposed as `memoryTraceNextDebug.graphCy` for outside stability assertions.
- R:
  - JNL: "the graph is totally unstable and changes every time i click a button" - root cause was the
    mount effect depending on `selectedId`/`labelIds`/selection-derived element sets, so every click
    destroyed the instance and re-ran a randomly-seeded physics layout. Stability is trust: the map is
    the user's mental model of their data and must not scramble under them.
  - JNL: "lines between related entries are curved unnecessarily causing noise" - the bezier bow
    carried no information; straight edges cut the ink without losing any semantics.
- A: Rejected keeping element-level evolves filtering with position pinning - class-based visibility is
  simpler and provably churn-free. Rejected a seeded random layout lib - deterministic initial
  positions + randomize:false achieves reproducibility with zero new dependencies.
- F: `memory-trace/client/src/GraphWorkspace.tsx`, rebuilt `memory_trace/static/react/*`.
- T: `tsc --noEmit` + `vite build` clean; live preview - selection click: **0/54 nodes moved**,
  zoom/pan stable; evolves tap-reveal: 1 edge revealed, 0 moved; labels "All": 54 labelled, 0 moved;

  console errors. Found pre-existing (not a regression): the all-dates overview API slice returns 60
