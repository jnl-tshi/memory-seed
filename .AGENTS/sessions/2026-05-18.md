# Session Log - 2026-05-18

## 2026-05-18 - CLI update command

- User approved `update` as the command keyword for refreshing reusable control-plane documents.
- Added `memory-seed update` as a CLI command that refreshes only the reusable seed files: `AGENTS.md`, `CLAUDE.md`, `GEMINI.md`, `.AGENTS/agent-rules.md`, and `.AGENTS/project-bootstrap.md`.
- `update` uses each control-plane file's `memory-system-version` YAML field to decide whether it is current.
- `update` backs up replaced control-plane files under `.AGENTS/backups/<timestamp>/`, restores missing reusable seed files, skips files already on the current control-plane version, and leaves generated project memory files such as `context.md`, `index.md`, `style.md`, and `sessions/` unchanged.
- Added `memory-seed update --dry-run` to preview the planned control-plane refresh without writing files.
- Bumped the Python package version to `1.3.1`; the reusable control-plane `memory-system-version` remains `1.3`.
